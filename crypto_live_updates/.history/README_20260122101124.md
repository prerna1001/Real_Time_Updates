# Crypto Live Updates

Real-time crypto price tracker using a shared Playwright TradingView scraper, a Node.js backend with WebSocket push, and a Next.js + Tailwind frontend.

## Stack

- pnpm workspaces (monorepo)
- Node.js + TypeScript backend (Fastify, Playwright, EventEmitter)
- Next.js (App Router) + Tailwind CSS frontend
- Shared TypeScript proto-style definitions for price events

## Getting Started

1. Install dependencies:

```bash
pnpm install
```

2. Start the backend server:

```bash
cd packages/server
pnpm dev
```

The server runs on http://localhost:4000 and exposes a WebSocket endpoint at ws://localhost:4000/prices.

3. In a separate terminal, start the frontend:

```bash
cd apps/web
pnpm dev
```

Then open http://localhost:3000 in your browser.

## Notes

- The TradingView DOM observer is currently mocked and emits random prices per symbol; you can replace the DOM selection logic in `packages/server/src/index.ts` with real selectors.
- The Next.js page is wired to the WebSocket stream and maintains a live event log.
